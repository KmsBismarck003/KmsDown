INICIO // SISTEMA UNITARIO

Bienvenido operador. Se ha establecido conexión con el manejador virtual de descargas.

Con este sistema puedes extraer directorios enteros evadiendo los límites convencionales de descarga. Utiliza las casillas de selección, guarda tus links favoritos y emplea la función Reparar para regenerar paquetes caídos.

MANUAL DE OPERACIÓN //

1. Escanear: Pega tu link y el sistema enumerará los bloques de datos listos para extracción.

2. Selección Fina: Destilda los checkbox de los paquetes que decidas omitir para optimizar caché local.

3. Pausa Individual: Ahora puedes congelar temporalmente un paquete usando [PAUS.] y continuarlo con [REAN.].

4. Autosave: Toca GUARDAR a un lado de escanear para salvar en local el link activo.
